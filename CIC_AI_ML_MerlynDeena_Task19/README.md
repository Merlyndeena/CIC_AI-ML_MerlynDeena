## 🧠 Topics Covered  

### ➕ NumPy Arithmetic Operations  
- Addition of arrays  
- Subtraction of arrays  
- Multiplication of arrays  
- Division of arrays  
- Element-wise operations without loops  

### 🔢 Scalar Operations  
- Add constant value to array  
- Multiply array with scalar  
- Subtract constant from array  

### ⚡ Vectorization  
- Performing operations on full arrays without loops  
- Faster than traditional Python loops  
- Cleaner and shorter code  
- Important in Machine Learning models  

### 📐 Mathematical Functions  
- Square root using `np.sqrt()`  
- Exponential using `np.exp()`  
- Logarithm using `np.log()`  
- Rounding using `np.round()`  
- Ceiling using `np.ceil()`  
- Floor using `np.floor()`  

### 📊 Aggregation Functions  
- Sum using `np.sum()`  
- Mean using `np.mean()`  
- Maximum using `np.max()`  
- Minimum using `np.min()`  

### ↕️ Axis Concept  
- Row-wise operations using `axis=1`  
- Column-wise operations using `axis=0`  
- Used in multi-dimensional arrays  

### 🔍 Comparison & Filtering  
- Compare array values using conditions  
- Filter values based on conditions  
- Useful for data cleaning and selection  

### 🌍 Real-World Thinking
