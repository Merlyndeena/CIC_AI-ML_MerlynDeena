---

## 🧠 Topics Covered  

### 📄 JSON (JavaScript Object Notation)  
- Data format used to store and exchange data  
- Similar to Python dictionary  
- Used in APIs and AI datasets  

### 🔄 JSON Operations  
- Convert Python dictionary → JSON (`json.dumps()`)  
- Convert JSON → Python dictionary (`json.loads()`)  
- Write JSON data to file (`json.dump()`)  
- Read JSON data from file (`json.load()`)  

### 📊 CSV (Comma Separated Values)  
- Used to store tabular data  
- Simple and widely used format  

### 🔄 CSV Operations  
- Create and write CSV file (`csv.writer()`)  
- Read CSV file (`csv.reader()`)  
- Read as dictionary (`csv.DictReader()`)  
- Print formatted output  

### 🧪 Virtual Environment  
- Isolated environment for Python projects  
- Avoids dependency conflicts  
- Helps in managing project libraries  

### ⚙️ Environment Setup  
- Create environment (`python -m venv myenv`)  
- Activate environment  
- Install packages using `pip`  

### 📦 requirements.txt  
- Stores project dependencies  
- Generated using `pip freeze`  
- Used to install dependencies (`pip install -r requirements.txt`)  

### 🌍 Real-World Thinking  
- JSON used in APIs and data exchange  
- CSV used in datasets and data storage  
- Virtual environments used in all professional projects  
- requirements.txt ensures reproducibility
