## 📌 Overview  
This document introduces NumPy, a powerful Python library used for numerical computations and array operations. It covers array creation, properties, indexing, slicing, and reshaping.

---

## 🧠 Topics Covered  

### 🔢 What is NumPy?  
- NumPy (Numerical Python) is used for numerical computations  
- Works with arrays efficiently  
- Faster and more memory-efficient than Python lists  

---

### ⚡ Why NumPy?  
- Faster execution (optimized in C)  
- Less memory usage  
- Supports vectorized operations  

---

### 📦 Array Creation  
- 1D arrays  
- 2D arrays  
- 3D arrays  
- Using `np.array()`  

---

### 📏 Array Properties  
- Shape → structure of array  
- Dimensions → number of axes (`ndim`)  
- Data type → type of elements (`dtype`)  

---

### 🧩 Special Arrays  
- Zeros matrix → `np.zeros()`  
- Ones matrix → `np.ones()`  
- Identity matrix → `np.eye()`  
- Range array → `np.arange()`  
- Random numbers → `np.random.rand()`  

---

### 🔍 Indexing & Slicing  
- Access elements in 1D array  
- Slice arrays  
- Access elements in 2D arrays  

---

### 🔄 Reshaping Arrays  
- Convert 1D → 2D using `reshape()`  
- Convert 2D → 1D using `flatten()` or `reshape()`  

---

### 🌍 Real-World Thinking  
- Data in AI/ML is stored as arrays  
- Images → 2D arrays  
- Videos → 3D arrays  
- NumPy is foundation for Pandas and ML libraries
