## 📌 Overview  
This document introduces data concepts, types of data, and basic data analysis techniques in Python. It also covers data cleaning, filtering, and transformation, which are essential steps in AI/ML workflows.

---

## 🧠 Topics Covered  

### 📊 What is Data?  
- Data is a collection of information  
- Examples: student marks, sales records, images, text  

---

### 🔢 Types of Data  
- Numerical Data → numbers  
- Categorical Data → labels (e.g., Male/Female)  
- Text Data → sentences or words  
- Image Data → pixel values  

---

### 🧾 Data Representation in Python  
- Using List → `[80, 90, 70]`  
- Using Dictionary → `{"name": "John", "marks": 80}`  
- Using List of Dictionaries → real dataset format  

---

### 📈 Basic Data Analysis  
- Calculate average using `sum()` and `len()`  
- Find maximum using `max()`  
- Count values using `len()`  

---

### 🧹 Data Cleaning  
- Removing unwanted or incorrect data  
- Example: removing `None` values  
- Helps improve data quality  

---

### 🔍 Data Filtering  
- Selecting required data based on conditions  
- Example: marks ≥ 50 (passed students)  

---

### 🔄 Data Transformation  
- Changing data into another format  
- Example: converting marks to percentage  
- Scaling and normalizing data  

---

### ⚡ Why NumPy is Needed  
- Python loops are slow for large datasets  
- NumPy provides faster and efficient operations  
- Used for handling large-scale data in AI/ML  

---
