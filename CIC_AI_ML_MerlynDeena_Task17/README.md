## 🧠 Topics Covered  

### 📚 Python Libraries  
- Collection of pre-written code  
- Helps perform tasks easily  
- Saves time and reduces code complexity  

### ➗ Math Module  
- Used for mathematical operations  
- Square root using `sqrt()`  
- Power using `pow()`  
- Rounding using `ceil()` and `floor()`  
- Constants like `pi` and `e`  

### 🎲 Random Module  
- Generate random numbers  
- Select random elements  
- Useful for testing and simulations  
- Used in ML experiments  

### 📅 Datetime Module  
- Get current date and time  
- Format dates using `strftime()`  
- Extract year, month, day  
- Used in logs and time-series datasets  

### 📊 Data Visualization  
- Represent data using graphs  
- Line graph using `plot()`  
- Bar chart using `bar()`  
- Helps in data analysis and reporting  

### 🌍 Real-World Thinking  
- `math` for calculations  
- `random` for simulations  
- `datetime` for time-based data  
- `matplotlib` for charts and insights
